# BUA Foods PLC — Equity Valuation & Investment Analysis

## Overview
Comprehensive financial analysis and valuation of BUA Foods PLC (NGX: BUAFOODS), a leading FMCG company in Nigeria. Originally completed in 2024. Updated and published March 2026.

## Methodology
- Discounted Cash Flow (DCF) analysis
- Financial statement forecasting (2024F–2028F)
- Net profit margin and gross profit trend analysis
- Upside/downside scenario modelling
- Closing cash balance and non-current asset projections

## Key Findings
| Metric | Value |
|--------|-------|
| Current Price | ₦394.9 |
| Target Price | ₦506.6 |
| Upside Potential | ~28% |
| Recommendation | **BUY** |
| Net Profit Margin | Stable at ~20% across forecast period |
| Outstanding Shares | 18,000,000 |
| Industry | FMCG |
